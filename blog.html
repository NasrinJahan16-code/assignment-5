What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents.
It represents the page so that programs can change the document structure, style, and content.

What is event delegation in the context of DOM, and why is it useful?

Event delegation is a technique where a parent element manages events for child elements using a single listener.
It improves efficiency by reducing the number of listeners and handles dynamically added elements automatically. This is achieved through
event bubbling, allowing events to propagate from child to parent elements.
